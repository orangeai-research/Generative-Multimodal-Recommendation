## Run the Code
1. Clone the repository
```bash
git clone 
```
2. Install dependencies
```bash
pip install -r requirements.txt
```
3. Run the code
```bash
cd GenMMRec/src
python main.py --model GenRecV1 --dataset baby
python main.py --model DiffMM --dataset baby
or
python run.py --config configs/diffmm.yaml
```
